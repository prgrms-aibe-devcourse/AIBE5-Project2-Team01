# 밋볼(MeetBall) MVP 요구사항 명세서

> **문서 목적:** 프로젝트 기반 팀원 모집 · 매칭 플랫폼 '밋볼'의 초기 핵심 기능(MVP)을 정의하고, 향후 와이어프레임, ERD, API 명세서 등의 기준을 마련합니다.

---

## 1. 프로젝트 개요

### 서비스명

- 가제: **밋볼**

### 기술 스택 (Technology Stack)
- **Framework**: Spring Boot 3.5.13
- **Language**: Java 17
- **Build Tool**: Maven
- **Database**: MySQL (운영), H2 Database (테스트/로컬)
- **ORM**: Spring Data JPA
- **Template Engine**: Thymeleaf
- **Other**: Lombok, Spring Boot DevTools, Spring Web

### 서비스 한 줄 소개: 프로젝트를 준비하는 사용자가 자신의 역할과 기술에 맞는 프로젝트를 찾거나 직접 팀원을 모집할 수 있는 매칭 플랫폼

## 2. 사용자 권한 구분
- **비회원:** 프로젝트 목록 및 상세 조회 (지원, 찜, 추천 기능 등은 클릭 시 로그인 유도)
- **회원(팀원):** 프로필 관리, 프로젝트 탐색/찜/지원, AI 맞춤 추천 조회, 리뷰 별점 확인
- **회원(팀장):** 프로젝트 등록/수정/삭제, 지원자 승인/거절, 문서 파일 관리, 리뷰 확인

---

## 3. 핵심 도메인별 기능 (MVP 범위)

### 🔑 3.1. 공통 및 인증 (Auth & Common)
- **소셜 로그인:** 구글 로그인 우선 지원
- **⭐️ 닉네임 커스텀 (신규 적용):** 최초 로그인 시 구글의 기본 이름 외에, 사용자가 **직접 새로운 닉네임과 필수 프로필 정보(직무, 스택 등)를 무조건 설정하도록 강제** 화면 이동
- **로그인 유도 & 권한 처리:** 비회원이 주요 기능(지원, 찜) 클릭 시 로그인 모달 노출하여 가입 유도
- **상태 처리:** 로딩 UI, 데이터 없음(Empty State), 비정상 접근 시 404/오류 페이지 적용

### 🏠 3.2. 홈 화면 (Home)
- **프로젝트 리스트 조회:** 리스트형/카드형 보기 전환 기능, 페이징 처리 지원
- **검색 및 필터:** 
  - 검색: 프로젝트명, 키워드, 기술스택 등
  - 필터: 포지션, 기술스택, 온/오프라인 여부 등
- **AI 큐레이션:** 로그인한 사용자의 직무와 관심사에 기반한 맞춤형 프로젝트 추천 (추천 사유 포함)

### 👤 3.3. 마이페이지 (My Page)
- **프로필 관리:** 내 정보(닉네임, 직무, 기술스택 등) 입력/수정 및 타인에게 보여줄 공개 범위 설정
- **내 기록 조회:**
  - **⭐️ 찜한 프로젝트 (신규 적용):** 홈 필터가 아닌 마이페이지 내 탭에서 모아보기 제공
  - 지원 현황 및 프로젝트 참여 완료/진행중 목록 확인
- **평가 확인:** 내가 받은 별점 리뷰 확인 (기본점수 기반 +- 형태)

### 🚀 3.4. 프로젝트 상세 및 관리 (Project)
- **상세 정보 확인:** 사용 기술, 참가 인원, 기간, 모집 조건 정보 확인 및 자유로운 댓글/질문
- **프로젝트 지원하기:** 희망 포지션 선택 후 내 프로필 기반으로 지원서(메시지 포함) 제출
- **팀장 기능 (관리):** 
  - 프로젝트 등록/수정/삭제 (진행 중 수정 제한, 삭제 불가 등 방어 로직 필요)
  - 마이페이지를 통해 지원자 목록 확인 및 승인/거절 처리
  - 프로젝트 결과물/첨부파일 스토리지 업로드 연동

## 4. MVP 제외 및 후순위 기능 (Phase 2)
- 구글 외 시스템 (카카오, 네이버 로그인) 및 일반 이메일 가입
- 실시간 채팅, 고도화된 타겟 알림, 상세 텍스트(주관식) 리뷰, 타인 프로필(링크드인 형태) 개인 페이지 방문
- 칭호, 배지 등 게이미피케이션(업적) 시스템
- 공모전, 교육, 이벤트 등 카테고리 확장 적용
