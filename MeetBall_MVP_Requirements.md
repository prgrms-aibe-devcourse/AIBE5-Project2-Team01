# 밋볼(MeetBall) MVP 요구사항 명세서 (최종본)

> **문서 목적:** 프로젝트 기반 팀원 모집 · 매칭 플랫폼 '밋볼'의 초기 핵심 기능(MVP)을 정의하고, 개발 및 UI/UX 설계의 기준을 확립합니다.

---

## 1. 프로젝트 개요

### 서비스명
- 가제: **밋볼 (MeetBall)**

### 서비스 한 줄 소개
- 프로젝트를 준비하는 사용자가 자신의 역할과 기술에 맞는 프로젝트를 찾거나 직접 팀원을 모집할 수 있는 **매칭 플랫폼**

### 기술 스택 (Technology Stack)
- **Framework**: Spring Boot 3.5.13
- **Language**: Java 17
- **Build Tool**: Maven
- **Database**: MySQL (운영), H2 Database (테스트/로컬)
- **ORM**: Spring Data JPA
- **Template Engine**: Thymeleaf

| 구분 | 프로젝트 탐색/조회 | 지원/찜/AI 추천 | 프로젝트 등록/관리 | 지원자 승인/거절 |
| :--- | :---: | :---: | :---: | :---: |

### 🔑 권한 제어 세부 방침 (보강)
1.  **UI 노출**: 팀장 전용 기능(수정/삭제/지원자 관리) 버튼은 **작성자 본인에게만 노출**됩니다.
2.  **접근 차단**: 비회원이 지원/찜 등 회원 전용 기능을 누를 경우 **로그인 유도 모달**을 띄웁니다.
3.  **URL 보안**: 비정상적인 경로로 타인의 관리 페이지에 접근 시 404 또는 접근 불가 페이지로 리다이렉트합니다.

## 3. 핵심 기능 요구사항 (MVP)

### 3.1. 인증 및 프로필 (Auth & Profile)
- **소셜 로그인**: 구글 계정 연동을 통한 빠른 가입 지원.
- **⭐️ 프로필 설정 강제화**: 최초 로그인 시 닉네임, 직무, 기술 스택 설정을 완료해야만 서비스 이용이 가능하도록 리다이렉트 처리합니다.
- **프로필 정보**: 닉네임, 프로필 이미지, 직무, 경력, 기술 스택, 자기소개, 링크 등.

### 3.2. 홈 화면 (Home)
- **목록 조회**: 카드형/리스트형 전환 지원, 페이징 처리.
- **검색 및 필터**: 제목, 기술 스택, 포지션, 온/오프라인 여부 기준 필터링.
- **AI 맞춤 추천**: 사용자의 프로필 역량을 분석하여 추천 사유와 함께 프로젝트 큐레이션 제공.

### 3.3. 마이페이지 (My Page)
- **기록 관리**: 내가 읽은/찜한/지원한/참여 중인 프로젝트 목록을 탭별로 구분하여 제공.
- **평판 시스템**: 받은 별점(1~5점)을 기반으로 한 **'밋볼 매너 온도'** 확인.
- **프로필 수정**: 내 정보 수정 및 공개/비공개 여부 설정.

### 3.4. 프로젝트 상세 및 관리 (Project)
- **상세 정보**: 모집 현황(포지션별), 기술 스택, 프로젝트 기간, 상세 설명, 팀장 정보 등.
- **댓글/질문**: 프로젝트 관련 질의응답 및 팀장 알림.
- **파일 관리**: 프로젝트 관련 문서(포트폴리오, 기획서 등) 업로드 및 다운로드.
- **팀장 전용 관리**:
    - 별도 관리 페이지 또는 상세 페이지 내에서 **지원자 목록 확인**.
    - 지원자에 대한 **[승인] / [거절]** 처리 및 인원 카운트 자동 연동.

---

## 4. 비기능 요구사항
- **응답성**: 검색 및 필터링 시 1초 이내 결과 출력.
- **안정성**: 첨부파일 업로드 시 UUID를 통한 파일명 충돌 방지 및 안전한 저장.
- **확장성**: 추후 채팅 및 다중 소셜 로그인 추가가 용이하도록 회원 구조 설계.

## 5. MVP 제외 및 후순위 기능

- 상세 텍스트(주관식) 리뷰 및 타인 프로필 페이지 방문
